** code block export template
   :PROPERTIES:
   :DATE:     2012-01-14
   :END:

The =org-babel-exp-code-template= variable may be customized to
control which information from code blocks is exported.
: org-babel-exp-code-template is a variable defined in `ob-exp.el'.
: Its value is "#+BEGIN_SRC %lang%flags\n%body\n#+END_SRC"
:
: Documentation:
: Template used to export the body of code blocks.
: This template may be customized to include additional information
: such as the code block name, or the values of particular header
: arguments.  The template is filled out using `org-fill-template',
: and the following %keys may be used.
:
:  lang ------ the language of the code block
:  name ------ the name of the code block
:  body ------ the body of the code block
:  flags ----- the flags passed to the code block
:
: In addition to the keys mentioned above, every header argument
: defined for the code block may be used as a key and will be
: replaced with its value.
:
: You can customize this variable.

For example the following will include code block names in export.
#+BEGIN_SRC emacs-lisp
  (setf org-babel-exp-code-template
        "=%name=\n#+BEGIN_SRC %lang%flags\nbody\n#+END_SRC")
#+END_SRC
